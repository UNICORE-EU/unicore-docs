vo.localServerURI

string

-

Can contain this, local server SAML identifier, to be used in SAML requests in PULL mode. If unset then DN identity is used for queries, created from the local server's credential. 

vo.pull.cacheTtl

integer number

``600``

Controls pulled attributes cache. Set to negative integer to disable the caching or to positive number - lifetime in seconds of cached entries. 

vo.pull.disableIfAttributesWerePushed

[true, false]

``true``

DEPRECATED 

vo.pull.enable

[true, false]

``true``

DEPRECATED 

vo.pull.enableGenericAttributes

[true, false]

``true``

If turned on, then not only the recognized UNICORE attributes are processed, but also all others, which can be used for authorization. 

vo.pull.verifySignatures

[true, false]

``true``

Additional security for the pulled assertions (except transport level which is always on) can be achieved by verification of signatures of the received assertions. The key which is used for verification must be present in the VO truststore. 

vo.pull.voServerAuthN.password

string

-

If certificate-based authentication to the VO server is disabled, you might be able to use username/password. This sets the password. 

vo.pull.voServerAuthN.username

string

-

If certificate-based authentication to the VO server is disabled, you might be able to use username/password. This sets the username. 

vo.pull.voServerURL

string

``localhost``

Full address (URL) of SAML VO service. Note that this server's CA cert must be present in the main truststore of the server to create the connection. 

vo.push.enable

[true, false]

``false``

DEPRECATED 

vo.truststore.[.*]

string `can have subkeys`

-

Properties starting with this prefix are used to configure validation of VO assertion issuers certificates. Trust anchors should contain only the trusted VO servers certificates. All options are the same as those for other UNICORE truststores. 

vo.unicoreAttribute.[.*]

string `can have subkeys`

-

Properties starting with this prefix are used to configure mappings of SAML attributes to UNICORE internal ones. 

vo.voServerURI

string

-

Identification URI of the VO service providing attributes for this source. Only attributes issued by this issuer will be honoured.

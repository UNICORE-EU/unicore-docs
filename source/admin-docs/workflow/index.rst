Workflow Service
****************

The Workflow service is a server component that supports submission
and execution of application workflows consisting of UNICORE jobs
and control constructs:

* :ref:`RESTful APIs <rest-api>`

* JSON :ref:`workflow description <workflow-description>`

* Full range of UNICORE user authentication options and AAI
  integration

      User Manual with detailed instructions and examples for using the Workflow service.

      Building the Workflow service distribution packages.
